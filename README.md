# VPet.ModMaker

《虚拟桌宠模拟器》MOD 制作器 0 门槛，打开就能用。支持所有游戏内容（动画，文本，效果皆可）。跟随游戏更新，出新内容，MOD 制作器也会跟随更新，立即上手制作吧！
